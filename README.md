SVG Pie Timer
================

An animated SVG pie timer using requestAnimationFrame

Current version: 0.9.1 released 28.08.2014 under MIT



Recommended use
---------------

```html
<script src="svg-pie-timer.js"></script>
<script>
svgPieTimer({
    element: svgElement,
    duration: 1000,
    loops: 2
});
</script>
```
